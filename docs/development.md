# Desenvolvimento

## Pré-requisitos

- Node.js 18+
- npm ou yarn

---

## Instalação

Clone o repositório:

git clone https://github.com/lucasmvieira94/sistema-le-jardin.git

---

Instale dependências:

npm install

---

Execute o projeto:

npm run dev

---

## Estrutura do Código

src/

components → componentes reutilizáveis  
pages → páginas do sistema  
hooks → hooks personalizados  
services → integração com backend  
utils → funções auxiliares

---

## Padrões utilizados

- componentes reutilizáveis
- hooks customizados
- organização por domínio
