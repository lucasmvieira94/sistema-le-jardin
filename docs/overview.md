# SENEXCARE - Visão Geral

## Sobre o Projeto

O SENEXCARE é uma plataforma web desenvolvida para apoiar a gestão administrativa e operacional de instituições de longa permanência para idosos (ILPI).

O sistema foi projetado com foco em simplicidade, baixo custo de infraestrutura e facilidade de uso, permitindo que instituições organizem suas informações e digitalizem processos institucionais.

O projeto foi inicialmente desenvolvido para o **Le Jardin Residencial Senior**.

---

## Problema

Instituições de cuidado de idosos frequentemente enfrentam desafios como:

- uso excessivo de registros em papel
- dificuldade na organização de informações
- ausência de sistemas integrados
- baixa visibilidade gerencial
- retrabalho administrativo

---

## Solução

O SENEXCARE oferece uma plataforma digital modular que permite:

- digitalizar processos administrativos
- organizar registros institucionais
- acompanhar rotinas operacionais
- gerar relatórios administrativos
- coletar feedback dos usuários

---

## Principais Módulos

- Gestão de Funcionários
- Registro de Ponto
- Escalas de Trabalho
- Prontuário de Residentes
- Controle de Rotinas (temperatura, fraldas)
- Relatórios
- Feedback do Sistema

---

## Público-alvo

- instituições de longa permanência para idosos
