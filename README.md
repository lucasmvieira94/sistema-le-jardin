# SENEXCARE
### Sistema de Gestão para Instituições de Longa Permanência para Idosos (ILPI)

SENEXCARE é uma plataforma web desenvolvida para apoiar a gestão administrativa e operacional de instituições de longa permanência para idosos. O sistema foi projetado para simplificar processos institucionais, organizar informações operacionais e fornecer ferramentas de apoio à tomada de decisão.

O projeto foi desenvolvido inicialmente para o **Le Jardin Residencial Senior** como parte de um projeto acadêmico de extensão do curso de **Análise e Desenvolvimento de Sistemas**.

---

# Objetivo do Projeto

O objetivo do SENEXCARE é fornecer uma solução digital simples, modular e acessível que permita às instituições:

- digitalizar processos administrativos
- organizar informações operacionais
- melhorar a gestão de equipes
- registrar atividades institucionais
- gerar relatórios gerenciais
- apoiar decisões baseadas em dados

A plataforma foi projetada considerando o contexto de instituições com recursos limitados e equipes com diferentes níveis de familiaridade tecnológica.

---

# Funcionalidades

O sistema possui uma arquitetura modular e atualmente contempla os seguintes módulos:

### Gestão de Funcionários
- cadastro e gerenciamento de funcionários
- controle de escalas
- consulta de registros individuais

### Registro de Ponto
- autenticação simples por PIN
- registro de entrada e saída
- consulta de histórico de registros

### Prontuário de Residentes
- registro de informações clínicas
- acompanhamento de ocorrências

### Monitoramento de Rotinas
- controle de temperatura
- controle de fraldas
- registros operacionais

### Relatórios
- relatórios administrativos
- exportação de dados
- visualização consolidada

### Feedback do Sistema
- coleta estruturada de opiniões dos usuários
- identificação de melhorias
- levantamento de requisitos para evolução da plataforma

---

# Arquitetura do Sistema

O SENEXCARE utiliza uma arquitetura moderna baseada em aplicações web.

### Frontend

- React
- TypeScript
- Vite
- Tailwind CSS
- shadcn-ui

### Backend

- Supabase
- PostgreSQL

### Gerenciamento de Dados

- React Query

### Recursos adicionais

- suporte a Progressive Web App (PWA)
- exportação de relatórios
- autenticação de usuários
- rotas protegidas

Essa arquitetura permite:

- baixo custo de infraestrutura
- escalabilidade
- facilidade de manutenção
- evolução modular do sistema
